# Sistema de Facturación (PHP & MySQL)

Sistema básico de facturación desarrollado en **PHP** y **MySQL**, enfocado en la gestión de usuarios mediante un panel administrativo protegido por inicio de sesión.

Este proyecto sirve como base para un sistema de ventas/facturación más completo, el cual será ampliado en futuras versiones.

---

##  Funcionalidades actuales

✔ Inicio de sesión con sesiones  
✔ Registro de usuarios (solo desde el panel)  
✔ Listado de usuarios  
✔ Hash de contraseñas con `password_hash()`  
✔ Protección de rutas mediante sesión  
✔ Diseño tipo panel administrativo  

---

## Control de acceso

- El sistema requiere iniciar sesión para acceder al panel.
- Los usuarios se crean desde el panel administrativo.
- Los usuarios finales solo inician sesión con las credenciales asignadas.
